# Stelch Secure

Partially Developed server application protect script.

## Prerequisites

* PHP 7 or above

### Installing

> This has been deprecated, it is not recomended to install.

Insert the following at the top of all files you wish to protect.

```php
<?php require("secure.inc.php"); ?>
```
